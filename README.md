# 国际商务谈判课程系统使用说明

## 一、这套系统是什么
这是一套以 `Obsidian + NotebookLM + Codex` 为核心的国际商务谈判课程运行系统。

它不是单一笔记库，而是一个完整的教学支持环境，包含：
- 知识库：沉淀谈判理论、方法、概念与价值导向
- 工作台：支持学生完成准备、信息收集、对手分析、策略讨论、模拟和复盘
- 模板库：提供可直接填写的任务单和记录卡
- 对象库：提供案例、对手画像、FAQ 和课堂活动
- 智能体层：提供 Codex 可执行的系统提示词和交互流程
- Canvas 可视化：帮助快速导航和理解系统结构

## 三、三种工具分别做什么
### 1. Obsidian
Obsidian 是主工作环境。

你在这里完成：
- 阅读知识笔记
- 打开工作台
- 填写模板
- 浏览案例、活动和 FAQ
- 使用 Canvas 进行全局导航

### 2. NotebookLM
NotebookLM 是课程资料阅读与知识提炼引擎。

它的作用是：
- 读取课程原始资料、教材、PDF、整理文档
- 提炼课程中的概念框架、方法、案例和复盘要点
- 为 Obsidian 知识库提供内容来源

### 3. Codex
Codex 是课程智能体执行层。

它的作用是：
- 根据工作台和模板协助学生完成任务
- 基于知识库给出结构化分析
- 按系统提示词执行价值导向与边界约束
- 帮助教师和学生进行模拟、复盘和案例分析

## 四、建议的日常使用路径
### 1. 课前准备
推荐路径：
1. 打开 [[02 工作台/谈判准备工作台]]
2. 填写 [[90 模板/谈判准备单]]
3. 如信息不足，进入 [[02 工作台/信息收集工作台]]
4. 如对手判断不清，进入 [[02 工作台/对手分析工作台]]
5. 最后进入 [[02 工作台/策略讨论工作台]]

### 2. 课堂中
推荐路径：
1. 从 [[03 对象库/案例库]] 选择案例
2. 参考 [[03 对象库/对手画像库]]
3. 使用 [[02 工作台/模拟演练工作台]]
4. 填写 [[90 模板/模拟演练脚本]]
5. 开展角色扮演、观察记录和裁判评估

### 3. 课后复盘
推荐路径：
1. 打开 [[02 工作台/复盘工作台]]
2. 填写 [[90 模板/谈判复盘单]]
3. 如涉及价值导向问题，补充 [[90 模板/价值引导记录卡]]
4. 将问题回流到准备、信息收集、对手分析或策略讨论环节

## 五、教师怎么用
教师可以把这套系统当成课程运行后台。

推荐方式：
- 用知识层讲概念
- 用对象库选案例、画像和活动
- 用工作台组织学生任务
- 用模板统一学生输出格式
- 用智能体层辅助答疑、模拟和复盘
- 用 FAQ 快速回应高频误区

## 六、学生怎么用
学生可以把它当成谈判任务助手。

推荐方式：
- 先看知识，不急着直接做策略
- 在工作台中按顺序推进
- 所有结论尽量写在模板里
- 复盘时不要只写“成功/失败”，要写原因和下一步改进

## 七、智能体怎么接入
如果要在 Codex 中使用这套系统，优先使用：
- [[20 智能体/国际商务谈判智能体系统提示词]]
- [[20 智能体/课堂交互流程]]
- [[20 智能体/学生交互模板]]

建议做法：
- 将系统提示词作为高优先级角色说明
- 让智能体优先依据工作台和模板回答
- 让智能体优先调用知识节点，而不是直接给空泛建议

## 八、这套系统的核心原则
- 先分析利益，再讨论策略
- 先创造价值，再分配价值
- 既分析己方利益，也分析对方利益
- 既关注个人目标，也关注组织责任、社会影响和国家利益
- 不使用欺诈式、霸凌式谈判策略
- 把课堂训练的重点放在判断力、沟通力、边界意识和复盘能力上
