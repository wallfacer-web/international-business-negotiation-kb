# 国际商务谈判智能体总览（网页版）

这页是 `国际商务谈判智能体总览.canvas` 的网页可展示版。它回答三个最核心的问题：

1. 我从哪里进入系统
2. 我现在是什么角色
3. 我当前处在知识、工作台、模板、对象库、智能体的哪一层

> [!info] 使用建议
> 如果你是第一次进入这个网站，先看这页。  
> 如果你已经知道自己的角色，请直接进入教师入口、学生入口或智能体入口。

## 系统主干

### 知识层

用于解释概念、方法、边界与价值导向。

建议入口：
- [[01 知识/谈判流程]]
- [[01 知识/利益分析]]
- [[01 知识/谈判策略]]

### 工作台层

用于按步骤推进任务。

主路径：
`准备 -> 收集 -> 对手分析 -> 策略 -> 模拟 -> 复盘`

建议入口：
- [[02 工作台/工作台导航]]
- [[02 工作台总览（网页版）]]
- [[02 工作台/谈判准备工作台]]
- [[02 工作台/对手分析工作台]]
- [[02 工作台/策略讨论工作台]]

### 模板层

用于统一课堂输出格式。

原则是：每个工作台都应落到对应模板，而不是只停留在口头讨论。

建议入口：
- [[90 模板/模板导航]]
- [[90 模板总览（网页版）]]
- [[90 模板/谈判准备单]]
- [[90 模板/对手分析卡]]
- [[90 模板/策略讨论单]]

### 对象库

用于提供案例、画像、FAQ 和课堂活动，把知识与任务变成可训练素材。

建议入口：
- [[03 对象库/案例库]]
- [[03 对象库/对手画像库]]
- [[03 对象库/学生常见问题库]]
- [[03 对象库/课堂活动库]]
- [[03 对象库总览（网页版）]]

### 智能体层

用于承接智能支持，但网页公开版不展示内部智能体提示词细节。  
在系统设计上，智能体应优先调用工作台与模板，而不是直接给空泛回答。

## 推荐第一视角使用法

### 教师视角

从 [[00 MOC/教师入口]] 进入，优先看案例库、课堂活动库、模拟演练与复盘。

### 学生视角

从 [[00 MOC/学生入口]] 进入，优先看工作台和模板。

### 智能体视角

从 [[00 MOC/智能体入口]] 进入，先定位工作阶段，再调用对应工作台与模板。
