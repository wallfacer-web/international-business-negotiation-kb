# 活动卡 02 BATNA 与底线计算练习

## 目标
训练学生把替代方案转化为 BATNA，再由 BATNA 推导底线。

## 形式
- 个人计算
- 小组对比

## 流程
1. 给出谈判失败后的多个替代选项。
2. 学生评估各选项可行性与价值。
3. 选出 BATNA。
4. 推导当前谈判底线。
5. 对比不同组的判断差异。
