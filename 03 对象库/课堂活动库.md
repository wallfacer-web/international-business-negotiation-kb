# 课堂活动库

## 用途
课堂活动库用于把知识卡片转化为可操作的课堂训练任务，帮助学生通过练习理解概念、通过模拟形成判断、通过复盘沉淀能力。

## 使用方式
- 可独立使用单个活动。
- 也可按“准备 -> 分析 -> 策略 -> 模拟 -> 复盘”的顺序串联。
- 推荐搭配案例库、画像库和模板库一起使用。
