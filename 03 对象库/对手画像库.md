# 对手画像库

## 用途
对手画像库用于沉淀典型谈判对象，帮助学生在不同案例中练习 [[01 知识/对手画像]]、[[01 知识/跨文化因素]] 和 [[01 知识/沟通技巧]] 的应用。这里的画像都是工作假设，不是固定标签。

## 使用方式
1. 先从画像库选一个对象。
2. 使用 [[90 模板/对手分析卡]] 做结构化分析。
3. 判断其利益、约束、权限、文化和沟通特征。
4. 再将结论转入 [[02 工作台/策略讨论工作台]]。

## 备注
对手画像库的目标是帮助学生形成“基于证据的工作假设”，而不是训练贴标签能力。每一张画像都应服务于更理性的分析、更克制的沟通和更规范的策略设计。
