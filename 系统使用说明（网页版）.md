# 系统使用说明（网页版）

这页是 `系统使用说明.canvas` 的网页可展示版。  
如果总览页负责告诉你“系统长什么样”，这页负责告诉你“这个系统怎么用”。

## 三种工具分别做什么

### Obsidian

主工作环境。

- 阅读知识笔记
- 打开工作台
- 填写模板
- 浏览对象库

### NotebookLM

资料提炼引擎。

- 读取教材、PDF、原始资料
- 提炼概念、案例和方法
- 为知识库补充内容来源

### Codex

执行层智能体。

- 协助分析
- 驱动工作流程
- 支持模拟与复盘

## 按角色使用

### 教师怎么用

- 从 [[00 MOC/教师入口]] 进入
- 先选案例和活动
- 用工作台组织学生任务
- 用模板统一输出
- 用复盘和价值引导做诊断

### 学生怎么用

- 从 [[00 MOC/学生入口]] 进入
- 先看知识，不急着直接做策略
- 再进入工作台推进任务
- 把结论写进模板
- 复盘时不要只写结果，要写原因和改进

### 智能体怎么用

- 从 [[00 MOC/智能体入口]] 进入
- 优先定位当前阶段
- 优先调用工作台与模板
- 再用知识和对象库补充解释

## 一句话理解

这个系统的使用顺序不是“先找答案”，而是：

`先理解结构 -> 再进入角色 -> 再按工作台推进 -> 最后用模板沉淀结果`
