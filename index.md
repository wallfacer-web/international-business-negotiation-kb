# 国际商务谈判课程门户

这是一个面向国际商务谈判课程建设、课堂运行与智能支持的在线知识门户。你可以从这里进入课程知识、工作台、模板、对象库和系统导航。

> [!info] 门户定位
> 这不是单一资料页，而是一套围绕 `知识学习 -> 任务推进 -> 模拟训练 -> 复盘改进` 组织起来的课程系统入口。

## 推荐使用路径

1. 先读 [[README]] 了解系统结构。
2. 再从 [[00 MOC/国际商务谈判智能体 MOC]] 进入主导航。
3. 教师从 [[00 MOC/教师入口]] 进入，学生从 [[00 MOC/学生入口]] 进入。
4. 课堂运行时优先使用工作台和模板，而不是直接跳到零散资料。

> [!tip] 使用建议
> 如果你是第一次进入，建议先看 `README` 和 `课程总导航`。  
> 如果你已经在上课，建议直接进入 `工作台导航` 或角色入口。
