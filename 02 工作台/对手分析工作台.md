# 对手分析工作台

## 用途
这个工作台用于把零散信息转化为结构化的 [[对手画像]]。目标不是给对方贴标签，而是形成一个基于证据、可不断修正的工作假设，为后续 [[策略讨论工作台]] 和 [[谈判模拟]] 提供依据。

## 推荐输入
- [[90 模板/信息收集卡]]
- [[01 知识/对手画像]]
- [[01 知识/跨文化因素]]
- [[01 知识/高语境与低语境]]
- [[01 知识/沟通技巧]]
- 与本案例相关的外部材料、课堂观察、对话记录

## 核心原则
- 所有结论都应区分 `事实`、`推测`、`建议`
- 先看利益与约束，再看风格与表现
- 对风格判断保持克制，不做人格定型
- 对跨文化差异做辅助判断，不用刻板印象替代分析

## 工作流程
### 1. 明确对象与权限
- 对方是谁
- 所属组织是什么
- 在谈判中承担什么角色
- 是否拥有最终决策权
- 是否需要向上级、法务、财务或团队汇报

### 2. 使用 ICAP 做基础判断
- `Interests`：对方真正想得到什么
- `Constraints`：对方受什么限制
- `Alternatives`：如果谈不成，对方还有什么选择
- `Perspectives`：对方如何理解风险、关系、时间和公平

### 3. 判断沟通与决策风格
- 更偏直接还是委婉
- 更重关系还是重效率
- 更重数据、程序还是互动感受
- 更偏个人拍板还是共识决策

### 4. 纳入跨文化因素
- 是否更接近高语境还是低语境沟通
- 是否重视面子、关系和会外沟通
- 是否存在明显的时间观差异

### 5. 输出对谈判的意义
- 适合怎样开场
- 适合怎样提问与确认
- 哪些议题宜先谈、哪些宜后谈
- 可能的误判风险是什么

## 产出要求
- 使用 [[90 模板/对手分析卡]]
- 必须写出判断依据
- 必须给出后续沟通建议
- 必须提示误判风险

## 检查问题
- 你的判断里哪些是事实，哪些是推测
- 你是否分析了对方的利益、约束和权限
- 你是否把文化差异误当成人格特征
- 你是否已经知道下一步该如何与此对象沟通
