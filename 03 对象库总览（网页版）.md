# 03 对象库总览（网页版）

这页是 `03 对象库总览.canvas` 的网页可展示版。  
对象库不是素材堆放区，而是课程训练素材层。它负责把知识卡片和工作台变成可调用的案例、对象、FAQ 和活动。

## 对齐原则

- 每类对象都要回连知识卡片
- 每类对象都要能进入工作台
- 每类对象尽量绑定对应模板
- 对象库服务于训练，不只是展示

## 推荐路径

`案例库 -> 对手画像库 -> 工作台 -> 模板 -> 模拟 / 复盘`

FAQ 作为快速答疑入口可以插入任意阶段。  
活动库作为课堂组织层串联全流程。
